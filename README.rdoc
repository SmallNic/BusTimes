BusTimes App

- Ruby version 2.2.0
- Rails version 4.2.1

This Bus Times application allows a user to search for the next times for a bus, add bus routes to a list of favorites and delete buses from a list of favorites.

When starting the application, the user has the following options:

- Search for a particular bus
- View the buses on his/her favorites list
- Delete buses from his/her favorites list
